# Online-Shop
- NodeJs should be installed in your system

## Technology
### Client: 
- EJS View Engine, Bootstrap 5.1
### Server: 
- ExpressJs, Mongoose, MongoDB, Bcrypt, PassportJs

## Run Locally
 To install external dependencies
```bash
npm install
```
 To start Project
```bash
npm start
```
