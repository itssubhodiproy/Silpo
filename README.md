# Online-Shop
- NodeJs should be installed in your system

## Technology
- ## Client: Ejs view engine, Bootstrap 5.1
- ## Server: ExpressJs, Mongoose, MongoDB, Bcrypt, PassportJs

## Run Locally
 To install external dependencies
```bash
npm install
```
 To start Project
```bash
npm start
```
